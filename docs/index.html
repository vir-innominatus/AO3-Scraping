    How to Read the Numbers

    • Score: Overall number used to rank similarity. It combines the cosine and overlap numbers. A higher value usually means a more reliable match.
    • Cosine: How similar the two works' kudos-reader audiences are, ignoring size. Higher means the same kinds of AO3 users tend to kudos both works. You can read more about it here: Cosine similarity overview.
    • Overlap: How many specific AO3 users gave kudos to both the work you picked and this recommendation. Bigger overlap means more direct shared readership.
    • Kudos Edges: How many individual kudos-user entries we have for this recommended work in the dataset. It's lower than the total number of kudos because of guest kudos from readers that don't have an AO3 account.
